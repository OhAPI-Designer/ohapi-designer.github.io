OhAPI Designer

A native macOS editor for OpenAPI 3.0 and 3.1 specifications. Combines the speed of a text editor with deep schema awareness—context-sensitive completions, inline validation, structured navigation, and a visual operations panel—all in one window. Supports JSON and YAML.

Intelligent Code Editor

Syntax highlighting with instant viewport rendering, line numbers, indent guides, auto-indentation, and breadcrumb navigation. See your cursor's position (paths › /users › get › responses › 200) and jump with a click. Adapts to macOS dark and light modes.

Context-Aware Completions and Snippets

Ctrl+Space for valid suggestions at your cursor—keys, values, references, filtered by context. 20+ adaptive snippet templates for operations, schemas, parameters. Tab through placeholders, add custom ones. Formats to JSON/YAML and OpenAPI version.

Inline Validation and $ref Navigation

Red wavy underlines for errors, amber for warnings, with hover tooltips. Full spec compliance including $ref resolution. Cmd+Click any $ref to jump to its definition—styled as blue links for instant recognition.

Operations Panel

Live table of endpoints with HTTP badges, paths, summaries. Tabbed views for details, parameters, interactive schema explorer (Referenced/Resolved), and example generator. Click anywhere to navigate to editor source. Updates as you type.

Built for macOS

Native Swift/AppKit—no Electron. JSON↔YAML conversion, new document wizard, external file detection, console log, customizable preferences, built-in manual. Tabs, autosave, App Sandbox. Apple Silicon and Intel.

How It Works

1. Open or Create

Use the wizard for new specs—pick version, format, paths—or open existing JSON/YAML files. Tabs for multiple documents.

2. Edit with Intelligence

Write with syntax highlighting, auto-complete (Ctrl+Space), snippets (type and Tab). Cmd+Click $refs, see validation underlines live.

3. Explore Operations

Operations panel lists endpoints. Click into details, schema trees, generated examples. Navigate back to editor seamlessly.

4. Convert and Export

Switch JSON↔YAML anytime. Save, autosave enabled. Console tracks events.

FAQ

Which OpenAPI versions and formats?

Full support for 3.0 and 3.1 in JSON and YAML. Templates and validation adapt automatically.

Is it native macOS?

Yes, pure Swift/AppKit. No web views or Electron. Optimized for Apple Silicon and Intel, with system appearance and sandbox.

How does validation work?

Inline underlines (red for errors, amber for warnings) with tooltips. Checks full spec including resolved $refs.

Can I customize?

Preferences for editor, indentation, colors in dark/light modes. Add custom snippet templates.
